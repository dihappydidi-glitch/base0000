# base0000 — Десятитысячно-единичная система счисления (base-10000 / B10K)

Система для представления и вычислений над **сколь угодно большими десятичными числами** без конвертации в машинный `int`. Все операции — поразрядные, над массивами цифр 0…9999.

## Формат чисел

### Целые числа

```
[знак]левая_половина : правая_половина
```

| Элемент | Описание |
|---------|----------|
| `знак` | `−` для отрицательных чисел |
| левая половина | 4-цифровые группы, разделённые `.` |
| `:` | ровно одно двоеточие |
| правая половина | 4-цифровые группы, разделённые `.` |

Обе половины всегда имеют **одинаковое число групп**.

**Модель:** цифры big-endian переплетаются между половинами:

```
BE = [L₀, R₀, L₁, R₁, …, Lₙ₋₁, Rₙ₋₁]
              
      2n−1           0
N = L₀·B    + … + Rₙ₋₁·B

где B = 10000, n = число пар, L₊ — левые группы, R₊ — правые
```

Чётные индексы BE → левая половина, нечётные → правая.

### Дробные числа

```
[знак]левая_половина_дроби : целая_часть , правая_половина_дроби
```

- Запятая `,` — десятичный разделитель (как в европейской нотации)
- Целая часть — в стандартном переплетающемся формате
- Левая и правая половины дроби переплетаются, образуя дробные разряды:

```
дробные_разряды = [L₁, R₀, L₂, R₁, L₃, R₂, …]

где L₊ — группы левой половины дроби (L₀ = 0000 — паддинг)
      R₊ — группы правой половины дроби (после запятой)
```

Итоговое число: `целая_часть . дробные_разряды`

## Примеры

### Целые

| Формат | Значение |
|--------|----------|
| `0000:0005` | 5 |
| `0001:0000` | 10 000 |
| `9999:9999` | 99 999 999 |
| `0000.0001:0000.0000` | 10 000 |
| `-0000:0005` | −5 |
| `0000.0000:0001.0000` | 100 000 000 |
| `0001.0000:0000.0000` | 1 000 000 000 000 |
| `9999.9999:0000.0000` | 9 999 000 099 990 000 |

**Арифметика:**

```
  0000:0005 + 0000:0003              =  0000:0008
  9999:9999 + 0000:0001              =  0000.0000:0001.0000
  0000:9999 × 0000:9999              =  9998:0001
  9999:9999 × 9999:9999              =  9999.0000:9998.0001
  9999.9999:0000.0000²               =  9998.9996.9998.0000:0002.0002.0001.0000
  0000:0002 ** 0000:0010             =  0000:1024  (2¹⁰)
  -0000:0007 // 0000:0003            =  -0000:0003  (Евклид: остаток 0000:0002)
  fact(0000:0005)                    =  0000:0120  (5!)
  isqrt(0000:0144)                   =  0000:0012  (√144)
  gcd(0000:0012, 0000:0008)          =  0000:0004
```

### Дробные

| Формат | Десятичное |
|--------|------------|
| `0000:0000,0000:0001` | 0,00000001 |
| `0000:0001,0000:0000` | 1,0 |
| `0000.5568.4490.0600.6222:0003,9330.0628.6154.2389` | 3,5568933044900628060061546222389… |

Дробные разряды из `0000.5568.4490.0600.6222:0003,9330.0628.6154.2389`:

```
Левая половина дроби: 0000.5568.4490.0600.6222
Правая половина дроби:                 9330.0628.6154.2389

Переплетение (L₁, R₀, L₂, R₁, L₃, R₂, …):
  5568.9330.4490.0628.0600.6154.6222.2389

Итого: 3,5568933044900628060061546222389…
```

```bash
git clone https://github.com/dihappydidi-glitch/base0000.git
cd base0000
python base10000.py              # тесты
python base10000.py --repl       # интерактивный калькулятор
```

## Арифметические операции

| Оператор | Описание |
|----------|----------|
| `a + b` | сложение |
| `a - b` | вычитание |
| `a * b` | умножение (столбиком) |
| `a / b` | целочисленное деление |
| `a // b` | целочисленное деление |
| `a % b` | остаток (Евклидов: 0 ≤ r < \|b\|) |
| `a ** b` | возведение в степень (b ≥ 0) |

**Дополнительные функции:**

| Функция | Описание |
|---------|----------|
| `fact(n)` | факториал |
| `gcd(a, b)` | наибольший общий делитель |
| `lcm(a, b)` | наименьшее общее кратное |
| `isqrt(a)` | целочисленный квадратный корень |
| `shift_left(a, k)` | a × 10000ᵏ |
| `shift_right(a, k)` | a ∥ 10000ᵏ |

## REPL-калькулятор

```bash
python base10000.py --repl
```

Интерактивный режим:

```
b10k> 9999:9999 + 0000:0001
  0000.0000:0001.0000
b10k> fact(0000:0005)
  0000:0120
b10k> x = 0001:0000
  x = 0001:0000
b10k> x * x
  0000.0000:0001.0000
```

Поддерживает:
- `+` `-` `*` `/` `//` `%` `**`
- `fact(n)` `gcd(a,b)` `lcm(a,b)` `isqrt(n)`
- Переменные: `имя = значение`
- Выход: Ctrl+C или `.exit`

## Внутреннее устройство

```python
@dataclass
class B10K:
    sign: int       # 1 или -1
    digs: List[int] # little-endian: digs[0] + digs[1]·B¹ + ... (B = 10000)
```

- **Little-endian:** младший разряд по индексу 0.
- **Trim:** без ведущих нулей (кроме самого нуля: `[0]`).
- **Ноль:** всегда `sign = 1`.
- **Все операции — поразрядно,** без конвертации в Python `int`.

### Алгоритмы

| Операция | Сложность | Метод |
|----------|-----------|-------|
| Сложение | O(N) | поразрядный перенос |
| Вычитание | O(N) | поразрядный заём |
| Умножение | O(N·M) | столбик |
| Деление | O(N·M) | уголок (школьный) |
| Степень | O(log₂(b) · M(N)) | бинарное возведение |
| Квадратный корень | итерационный | метод Ньютона |
